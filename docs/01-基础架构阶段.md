# 第一阶段：基础架构搭建

> 前置条件：请先阅读 00-全局架构约定.md

## 1 阶段目标

搭建可运行的前后端骨架，实现用户登录认证，完成数据库接入。

## 2 开发任务

### 2.1 后端基础框架

#### 任务 1：初始化 Go 项目

- 创建 server/cmd/server/main.go
- 引入 Gin、GORM、Zap、Viper 依赖
- 实现配置加载（读取 configs/config.yaml）
- 实现日志初始化（Zap）

#### 任务 2：数据库接入

- TDengine 接入：使用 taosdata/taos-go 驱动，封装 DAO 层
- 达梦 DM8 接入：使用 GORM DM8 驱动，封装 DAO 层
- Redis 接入：使用 go-redis/redis 封装缓存层
- 编写数据库连接池配置

#### 任务 3：JWT 认证中间件

- 实现 /api/v1/auth/login 接口（用户名/密码登录）
- 实现 /api/v1/auth/userinfo 接口（获取当前用户信息）
- JWT Token 生成与验证
- 认证中间件（从 Header 提取 Token 并验证）
- RBAC 权限中间件（管理员/运维/只读）

#### 任务 4：用户管理 CRUD

- GET /api/v1/users - 用户列表
- POST /api/v1/users - 创建用户
- PUT /api/v1/users/:id - 更新用户
- DELETE /api/v1/users/:id - 删除用户
- 初始化默认管理员账号（admin/admin123）

#### 任务 5：统一响应格式

- 封装统一的 JSON 响应结构 { code, message, data }
- 错误码定义（10001=参数错误, 10002=认证失败, 10003=权限不足, 20001=数据库错误）
- 全局错误处理中间件

### 2.2 前端基础框架

#### 任务 6：Vue 3 项目初始化

- 使用 Vite 创建 Vue 3 + TypeScript 项目
- 安装 Element Plus、ECharts、Axios、Pinia、Vue Router
- 配置 ESLint + Prettier

#### 任务 7：布局框架

- 左侧导航栏（深色主题，可展开/收起）
- 顶部工具栏（面包屑、全局搜索、时间范围选择器、刷新按钮、用户头像下拉）
- 主内容区（router-view）
- 登录页（用户名/密码输入框 + 登录按钮）

#### 任务 8：路由配置

- /login - 登录页
- /dashboard - 首页（占位）
- /monitor/traffic - 流量监控（占位）
- /topology - 拓扑（占位）
- /trace - 追踪（占位）
- /profiling - 剖析（占位）
- /forensics - 回溯（占位）
- /probe - 探针管理（占位）
- /alert - 告警管理（占位）
- /settings - 系统设置（占位）
- 路由守卫：未登录跳转 /login

#### 任务 9：Axios 封装

- 请求拦截器：自动添加 JWT Token
- 响应拦截器：统一错误处理（401 跳转登录）
- API 接口定义文件：api/auth.ts、api/user.ts

#### 任务 10：登录功能联调

- 前端登录页 → 调用后端 /api/v1/auth/login
- 存储 Token 到 localStorage
- 登录成功跳转首页
- 退出登录清除 Token

### 2.3 部署脚本

#### 任务 11：一键安装脚本

- deploy/install.sh：检测环境 → 安装依赖 → 编译 → 构建前端 → 初始化数据库 → 配置 systemd → 启动服务
- deploy/ecosystem.config.js：PM2 配置
- deploy/cloud-flow.service：systemd service 文件

## 3 验收标准

- [ ] 后端服务可通过 systemctl start cloud-flow 启动
- [ ] systemctl status cloud-flow 显示运行中
- [ ] 前端访问 http://\<server-ip\>:3000 显示登录页
- [ ] 使用 admin/admin123 登录成功，跳转到首页
- [ ] 用户管理 CRUD 功能正常
- [ ] JWT Token 过期后自动跳转登录页
- [ ] PM2 list 显示 cloud-flow-server 进程

## 4 数据库初始化 SQL

### DM8 建表语句

```sql
CREATE TABLE users (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(64) NOT NULL UNIQUE,
    password_hash VARCHAR(256) NOT NULL,
    role VARCHAR(32) NOT NULL DEFAULT 'viewer',
    email VARCHAR(128),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

INSERT INTO users (username, password_hash, role) VALUES
('admin', '$2a$10$xxx...', 'admin');
```
