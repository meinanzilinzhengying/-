# eBPF 云内流量监测平台 - 全局架构约定

## 1 项目概述

- 基于 eBPF 的云内流量监测平台
- 三层架构：探针 → 边缘节点 → 中心服务器
- 前端 Vue 3 + 后端 Go + 探针 Go+eBPF
- 国产数据库：TDengine（时序）+ 达梦 DM8（关系型）
- 信创兼容：麒麟 V10 / 统信 UOS

## 2 技术栈约定

### 后端

- 语言：Go 1.21+
- HTTP 框架：Gin
- RPC：gRPC + Protobuf
- WebSocket：gorilla/websocket
- ORM：GORM
- 时序数据库：TDengine（驱动：taosdata/taos-go）
- 关系型数据库：达梦 DM8（GORM DM8 驱动，兼容 PostgreSQL 协议）
- 缓存：Redis（go-redis/redis）
- 日志：zap
- 配置：viper
- 进程管理：PM2（通过 pm2 启动所有 Go 服务）
- 系统服务：systemctl（通过 systemd unit file 管理 PM2 进程）

### 前端

- 框架：Vue 3 + TypeScript + Vite
- UI：Element Plus
- 图表：ECharts 5
- 拓扑图：AntV G6
- 火焰图：d3-flame-graph
- 状态管理：Pinia
- 路由：Vue Router 4
- HTTP：Axios
- 包管理：pnpm

### 探针

- 语言：Go + CGO（调用 eBPF C 库）
- eBPF 库：libbpf-go (cilium/ebpf)
- 部署：K8s DaemonSet / 裸机二进制

## 4 部署约定

### PM2 进程管理

所有 Go 服务通过 PM2 统一管理。ecosystem.config.js 配置：

```javascript
module.exports = {
  apps: [
    {
      name: 'cloud-flow-server',
      script: './server/cmd/server/main.go',
      interpreter: 'go',
      args: 'run',
      cwd: '/opt/cloud-flow',
      env: { NODE_ENV: 'production' },
      log_date_format: 'YYYY-MM-DD HH:mm:ss',
      error_file: '/var/log/cloud-flow/server-error.log',
      out_file: '/var/log/cloud-flow/server-out.log',
      max_memory_restart: '500M',
      restart_delay: 5000,
      max_restarts: 10
    }
  ]
};
```

### systemd 服务

```ini
[Unit]
Description=Cloud Flow Platform
After=network.target tdengine.service redis.service

[Service]
Type=forking
User=root
WorkingDirectory=/opt/cloud-flow
ExecStart=/usr/bin/pm2 start /opt/cloud-flow/deploy/ecosystem.config.js
ExecStop=/usr/bin/pm2 stop all
ExecReload=/usr/bin/pm2 reload all
PIDFile=/run/cloud-flow.pid
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

### 一键安装脚本 install.sh 功能

1. 检测操作系统和内核版本
2. 安装依赖（Go 1.21+、Node.js、pnpm、PM2、TDengine、Redis）
3. 编译后端、边缘节点、探针
4. 构建前端
5. 初始化数据库（创建 DM8 表、TDengine 超级表）
6. 配置 systemd 服务并启用
7. 启动所有服务
8. 输出访问地址和管理员默认密码

## 5 API 约定

- 统一前缀：/api/v1
- 统一响应格式：`{ "code": 0, "message": "ok", "data": {...} }`
- 认证方式：JWT Token（Header: `Authorization: Bearer <token>`）
- 分页参数：page（默认1）、page_size（默认20）
- 时间参数：start_time、end_time（Unix 时间戳或 RFC3339）

## 6 数据库约定

### TDengine

- 数据保留：30 天自动过期
- 超级表：每个探针是一个子表
- 采集频率：默认 10 秒

### 达梦 DM8

- 字符集：UTF-8
- 兼容 PostgreSQL 协议
- GORM 驱动配置

## 7 配置文件约定

所有组件使用 YAML 配置文件，格式统一：

```yaml
app:
  name: cloud-flow-server
  version: 1.0.0

server:
  http:
    host: 0.0.0.0
    port: 8080
  grpc:
    host: 0.0.0.0
    port: 9090

database:
  tdengine:
    host: localhost
    port: 6030
    user: root
    password: taosdata
    db_name: cloud_flow
  dm8:
    host: localhost
    port: 5236
    user: SYSDBA
    password: your_password
    db_name: cloud_flow
  redis:
    host: localhost
    port: 6379
    password: ""
    db: 0

log:
  level: info
  file: /var/log/cloud-flow/server.log
  max_size: 100  # MB
  max_backups: 10
  max_age: 30    # days

jwt:
  secret: your-jwt-secret-change-in-production
  expire: 24h    # Token 过期时间
```
